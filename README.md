# Personal Expense Tracker

A modern, responsive Personal Expense Tracker built with React (Vite) and Tailwind CSS.

## Features

- Add expense with title, amount, category, and date
- Display expenses in a responsive table
- Save and load expense data from `localStorage`
- Delete expense entries
- Summary cards for total spent and entry count

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```
